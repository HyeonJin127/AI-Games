
# Maze RL Project (Skeleton)

This is a minimal project skeleton for your maze-based RL game:

- `env_maze.py`: environment (maze, missions, items).
- `ai2_policies.py`: AI2 rule-based bot.
- `train_qlearning.py`: tabular Q-learning training.
- `train_dqn.py`: DQN training with PyTorch.
- `run_random_demo.py`: random or simple demo run.
- `plot_rewards.py`: reward visualization.

All files are placeholders; fill them with your own code.
